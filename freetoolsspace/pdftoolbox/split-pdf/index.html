What this tool does

Split PDF pages online for free by entering page ranges and creating a smaller browser-generated PDF. Use it when a source file contains extra pages, multiple forms, long chapters, or scans that need to be separated before submission.

The tool is built for quick extraction tasks: choose the PDF, type the pages you want, create the new file, and verify that the output contains only the selected pages.

How to split a PDF by page range

new PDF = selected pages copied from the source PDF in the requested range order
  1. Open the source PDF first and note the page numbers you actually need.
  2. Enter a range such as 1-3, a list such as 2,4,6, or a combined range such as 1-2,5.
  3. Generate the smaller PDF and compare it with the original so you do not accidentally remove a required page.
  4. Use Merge PDF afterward if you need to combine the extracted pages with another document.

Common use cases

  • Extract one chapter
  • Send selected forms
  • Remove extra scan pages
  • Create a sample PDF
  • Separate receipts
  • Trim upload size

Splitting is useful when a file is too large for an upload limit or when a recipient should only see a specific page range. It also helps clean up scanner output by removing covers, duplicates, and blank pages before sharing.

Split PDF FAQ

Is Split PDF free?

Yes. It is free and browser-based, with no account required for the core split workflow.

What page range format can I use?

Use simple ranges such as 1-3, individual pages such as 2,4,6, or a combination such as 1-2,5. Check the source PDF page numbers first.

Can splitting reduce PDF file size?

Often yes. Removing unneeded pages usually creates a smaller output file, although image-heavy pages can still be large.

Should I verify the output?

Yes. Always review important PDFs before submitting, printing, or deleting the original file.
