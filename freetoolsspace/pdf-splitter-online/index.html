How to split a PDF online

Upload a PDF, type the pages you want to keep, then generate a new file. This is useful when a form, ebook, invoice packet, or scanned document contains extra pages. The splitter is frontend-only and works well for simple extraction jobs where you need a smaller PDF without installing desktop software.

For best results, keep ranges simple and check the downloaded file before sending it. If your PDF is encrypted or unusually large, your browser may not be able to process it.

FAQ

Can I split a PDF without uploading it?

Yes. The file is read in your browser and the split PDF is created locally.

What page range formats are supported?

Use ranges like 1-3, single pages like 5, or a mixed list like 1-2,4,7.

Does this remove pages from the original PDF?

No. It creates a new PDF copy containing only the pages you selected.

Is there a size limit?

Your browser memory is the practical limit; very large PDFs may take longer.
