About this URL encoder and decoder

Use this free URL encoder when a link, query parameter, redirect URL, UTM value, search term, or API request contains spaces, ampersands, accented characters, emoji, or other symbols that need to be represented safely in a URL. The encoder converts those characters into percent-encoded text such as %20 for a space.

The decoder does the reverse: paste a percent-encoded string to read the original text again. This is useful when debugging campaign links, analytics URLs, webhook payloads, OAuth redirects, or copied browser addresses that contain encoded characters.

Encode query strings safely

When building URLs, encode the value that belongs inside a query parameter instead of blindly encoding the entire address. For example, keep https://example.com/search?q= readable, then encode the search phrase that follows it. This helps preserve separators like ?, &, /, and =.

The tool runs locally in your browser, so temporary links and draft parameters are not sent to a server for the core encode/decode action. Always test final production links after encoding, especially if they include nested URLs or redirect parameters.

URL encoder FAQ

What is URL encoding used for?

URL encoding converts spaces, symbols, Unicode characters, and reserved characters into percent-encoded text that can be placed safely in URLs and query strings.

Should I encode a whole URL or only a parameter?

Usually encode only the query value or parameter value. Encoding the whole URL can turn structural characters like slashes and question marks into text and make the link unusable.

Can this decode percent-encoded text?

Yes. Paste text such as hello%20world and click Decode URL to restore readable characters.

Is my URL text uploaded?

No. The encoder and decoder run with client-side JavaScript in your browser.
