How this 100KB compressor works

The tool draws your image to a browser canvas, optionally limits maximum width, then exports a compressed JPEG, WebP, or PNG. It tries lower quality levels until the output approaches your target size. Exact 100KB output is not guaranteed because image complexity, dimensions, and format limits all affect final file size.

Best results

If a photo is still too large, reduce the maximum width first, then lower quality. For ID photos, application forms, and profile uploads, compare the preview with the original before downloading.

Common uses include school forms, job applications, avatar uploads, marketplace listings, and government portals that reject images above a fixed size limit.

FAQ

Can every image be compressed to exactly 100KB?

Not always. Very large or detailed images may need resizing or visible quality reduction to approach 100KB.

Does compression reduce quality?

Lossy JPEG and WebP compression can reduce quality. The preview helps you choose a practical balance.

Are images uploaded to a server?

This tool uses browser canvas processing, so the selected image is processed locally in your browser.

Which formats work best?

JPEG and WebP usually compress photos well. PNG is better for crisp graphics but can stay larger.
